ng-suggest

AngularJS module to use OpenSearch Suggestions and SeeAlso

ng-suggest is a module for AngularJS to facilitate the use of OpenSearch Suggestions and SeeAlso links in web application.

The latest release of ng-suggest is version 0.1.0. Source code and issue tracker can be found at https://github.com/gbv/ng-suggest.

Download

The current release of ng-suggest is version 0.1.0, including:

  • ng-suggest.min.js: minified version for production
  • ng-suggest.js: original source code for development

Getting started

For details and examples see the documentation of services and directives provided by ng-suggest.

Customization

To support APIs that do not strictly follow OpenSearch Suggest specification, the OpenSearchSuggestions service can be customized with a transformation function. The suggest-typeahead directive contains a default template that can be replaced or styled with CSS.
