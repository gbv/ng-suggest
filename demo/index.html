Typeahead via OpenSearch Suggest

OpenSearch defines a set of rules for the open return of search queries. The format is supported by several big sites, including Wikipedia, and by all major browsers. Suggest services are used for providing autocomplete functions for search queries, and are prevalent in todays internet experience. This demo showcases the implementation of some OpenSearch Suggest services with AngularJS.

geonames

Geonames (geonames.org/) provides a lot of data for geographic places. This includes coordinates and markings on a map, as well as other contextual information.
